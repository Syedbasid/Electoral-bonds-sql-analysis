## Project Overview
This project analyzes India's Electoral Bonds data using SQL to uncover insights about
donors, political parties, cities, branches, and unaccounted funding.

The analysis answers 40+ real-world business and policy questions using complex SQL queries.

---

## Objectives
- Analyze donor spending patterns
- Identify political parties receiving highest and lowest funds
- Detect unaccounted or anonymous bond transactions
- Perform year wise, city wise, and denomination wise analysis
- Evaluate bank branch and employee involvement

---

## Tools & Technologies
- SQL (MySQL / PostgreSQL)
- CSV datasets
- GitHub for version control

---

## Key Analysis Performed
- Total funds spent on electoral bonds
- Year wise and month-wise bond purchases
- Top and least donors & political parties
- City wise bond purchase and encashment trends
- Bonds without donor information
- Bank employee performance analysis

---

## Sample SQL Questions Answered
- Which company bought the highest number of bonds?
- Which political party received the highest donations?
- How much unaccounted money exists?
- Which city had the highest bond purchases?
- Which employee issued the highest number of bonds?

---

## Key Learnings
- Advanced SQL querying (JOINs, CTEs, subqueries)
- Real world financial and political data analysis
- Writing analytical queries with business context
